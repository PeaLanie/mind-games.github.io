Intermediate

Now you understand adding numbers with 2 digits which all end with zeros. Now what if the digits actually end with a real number instead of a zero? It's actually easy but you have to store some numbers in your short term memory which you will then recall later.

The Theory:

(Think you already now how to add natural numbers? Jump straight to the game)

Now we are going to look at numbers that end with a 5 and a 0. In this mode you will sometimes come across a 5 standing alone.

So, the theory here is to first add the second digits in order to simplify the list. When you add the second digits, you will have to store the sum in your short term memory and the space they occupied will be replaced with 0.

Once you finish this step, you will find that the list start to look familiar...it becomes a Beginner Mode kind of list, if you're familiar with this list; then you know how to move on from here. If not, I strongly recommend you start from the Basics
Enough with the theory; lets be practical. Check out some examples below.

How to add up a list of real numbers between 5 and 95 with increments of 5?

Example #1

Below is a list of numbers that either end with a 0 or a 5. Look at the adjacent steps to understand how to add these numbers.

  1. First of all, add up the 5's and store the answer in your short term memory. Now the list should look like a Beginner Mode kind of list.

  2. From there, temporarly drop the zeros. Now the list becomes a Basic Mode kind of list.

  3. Add that list and undrop the zero back to the sum. Now add the number that you stored in your memory, in this case it is 15.

  4. The final answer should be 215

Note: If you are not familiar with step 2, I recommend you go back to the Basics
